# 📊 Netflix Content Analysis using Power BI

## 📌 Overview
This project analyzes Netflix movies and TV shows dataset to uncover insights related to genres, content ratings, and growth trends.  
An interactive Power BI dashboard is created to visualize patterns and support data-driven decisions.

---

## 🛠 Tools & Technologies
- Power BI  
- Excel / CSV  
- Data Cleaning & Visualization  

---

## 📊 Key Insights
- 🎬 Movies dominate the platform compared to TV Shows  
- 🎭 Drama is the most popular genre on Netflix  
- 📈 Content production increased significantly after 2015  
- 🔞 TV-MA is the most common content rating  

---

## 💡 Business Use Case
This dashboard helps streaming platforms:
- Understand audience content preferences  
- Identify trending genres  
- Plan future content strategy based on historical data
